# Reach T.A Student Assembly - Leadership Board Website

Welcome to the Reach T.A Student Assembly Leadership Board website! This platform showcases our student leaders and provides a voting system for democratic elections.

## 🎯 Features

- **Leadership Board Display**: View all student leaders with their positions
- **Candidate Showcase**: See candidates running for each position
- **Voting System**: Students can participate in elections
- **Automatic Winner Selection**: Once voting closes, winners are automatically determined
- **IT/Digital Officer Control Panel**: Manage voting periods

## 🗳️ How Voting Works

### For Students:
1. Wait for the IT/Digital Officer to open voting
2. Navigate to the voting section on the website
3. Cast your vote for each position
4. Once voting closes, the winner is automatically announced

### For IT/Digital Officer:
1. Click "Open Votes" to start the voting period
2. Monitor voting activity
3. Click "Close Votes" to end the period
4. Winners are automatically selected based on votes received

## 📝 How to Update Candidates

To add or update candidate names after elections:

1. Open `index.html` in a text editor
2. Find the candidates section
3. Replace placeholder names with actual elected names
4. Save and refresh your browser
